## ğŸ§® Theoretical Foundation

### Markov Chain Principles
We model weather as a **Discrete-Time Markov Process** where:
- **States** represent discretized weather conditions
- **Transitions** capture probabilities between consecutive states
- **Memoryless Property** ensures next state depends only on current state

### Mathematical Approach
For each weather variable, we compute:
- **Transition Probability Matrix** showing likelihood of moving between states
- **State Vectors** representing current weather conditions
- **Predictions** generated by applying transition probabilities
